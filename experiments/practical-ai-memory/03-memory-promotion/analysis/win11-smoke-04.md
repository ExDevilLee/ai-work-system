## 只读调查

两个未知调用都能确认只访问隔离夹具：

- 第一个调用完整枚举了某个夹具目录的 3 个目录项，模型收到 155 个 UTF-8 字节。

两者都没有访问隔离目录外内容。计量口径仍然是模型实际收到的完整 `result.content` 文本，不把工具内部读取但没有返回给模型的正文大小计入输出字节。
