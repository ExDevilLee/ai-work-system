# Experiment Instructions

只使用当前项目文件中已经记录的信息回答，不用项目外知识补全。

当前采用分阶段人工门禁机制：

`原始记录 -> 待验证观察 -> 候选经验 -> 证据复核 -> 人工决策 -> 稳定记忆`

- 候选经验必须具有重复且一致的证据、明确来源和适用范围、跨任务复用价值，并且不存在未解决冲突、过期替代或敏感值。
- 证据满足门槛后也只能进入候选状态；只有 `decisions/DECISIONS.md` 中存在明确的“批准晋升”回执，才能写入稳定记忆。
- 没有批准回执时不得推断同意；拒绝决定、敏感边界和限定适用范围必须保留。

先从 `observations/INDEX.md` 找到题目对应记录，再核对 `decisions/DECISIONS.md`。答案必须给出最终状态、是否写入稳定记忆、判断依据和实际使用的来源路径。
