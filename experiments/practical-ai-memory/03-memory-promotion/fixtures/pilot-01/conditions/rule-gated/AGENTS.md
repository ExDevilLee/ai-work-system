# Experiment Instructions

只使用当前项目文件中已经记录的信息回答，不用项目外知识补全。

当前采用规则门禁机制。只有同时满足以下条件，才自动晋升稳定记忆：

1. 至少有两次独立且方向一致的证据。
2. 来源、适用范围和当前状态可以追溯。
3. 对未来多个同类任务有复用价值。
4. 没有未解决的冲突，也没有被新版本替代。
5. 不包含账号、令牌、本机路径或其他不必要的敏感值。

未满足条件时，保留为待验证观察、等待冲突解决，或拒绝晋升。先从 `observations/INDEX.md` 找到题目对应记录；需要确认既有人工决定时读取 `decisions/DECISIONS.md`。答案必须给出最终状态、是否写入稳定记忆、判断依据和实际使用的来源路径。
